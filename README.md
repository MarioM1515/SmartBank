# SmartBank 💳

SmartBank je demo web aplikacija napravljena u HTML, CSS i JavaScript-u.  
Cilj projekta je da prikaže osnovne funkcionalnosti klijentskog interfejsa i serverske logike.

## 🚀 Live Demo
Aplikacija je dostupna preko GitHub Pages:  
[SmartBank](https://MarioM1515.github.io/SmartBank/)

## 📂 Struktura projekta
- `client.html` – glavna stranica aplikacije  
- `style.css` – stilovi za izgled sajta  
- `script.js` – JavaScript logika  
- `server/` – serverski deo projekta  
- `SmartBank.png` – logo/slika projekta  

## ⚡ Tehnologije
- HTML  
- CSS  
- JavaScript
